# Proyecto de Axel Pobian.

## Clon no funcional de "Whatsapp".

Maquetación estilo WhatsApp 2. Descripción breve
Este proyecto consiste en la maquetación utilizando HTML5 y CSS3 de una interfaz inspirada en WhatsApp Web.
Qué es: Una réplica visual de la aplicación de mensajería para practicar diseño web moderno.
Alcance: Se reprodujo tanto la barra lateral (sidebar) con la lista de contactos, como el panel principal de chat que incluye el encabezado, el área de mensajes con scroll y la barra de entrada de texto (footer).
Tecnologías: Se utilizó HTML semántico para la estructura y CSS avanzado para el diseño, sin el uso de JavaScript. 3. Estructura del proyecto
/index.html: Estructura principal del documento.
/styles.css: Hoja de estilos con variables y diseño responsivo.
/assets/: Carpeta que contiene el fondo de WhatsApp (fondo-wpp.png) e iconos de perfil. 4. Cómo visualizarlo
Para ver el proyecto, simplemente descarga los archivos y abre el archivo index.html en cualquier navegador moderno (Chrome, Firefox, Edge). 5. Decisiones de diseño / Consideraciones
Flexbox: Se utilizó display: flex y flex-direction: column en el panel principal para garantizar que el encabezado y el footer se mantengan fijos mientras los mensajes ocupan el espacio sobrante con scroll.
Medidas Dinámicas: Se aplicó height: 100% al contenedor principal para que la aplicación ocupe siempre el total del alto de la pantalla del usuario.
Estética: Se utilizaron variables de CSS para manejar la paleta de colores grises y verdes característicos de la aplicación, y se implementó un fondo con background-repeat para imitar el patrón original de la interfaz. 6. Trabajo con Git
El proyecto fue gestionado mediante un repositorio de Git, realizando commits frecuentes y descriptivos que documentan la evolución del desarrollo:
Maquetación inicial de la estructura.
Corrección de errores de anidamiento en el footer y mensajes.
Ajustes finales de posicionamiento y visualización.
